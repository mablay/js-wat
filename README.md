# JS-WAT
JavaScript - What!?

js-wat evaluates and inspects expressions.
Meant for educational / entertainment purpose.

## Node
Edit `/src/expressions.js` at your leisure.

    npm start

## Browser
You can edit the expressions interactively.

    npm run build
    npm run web
